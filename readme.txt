Wafer Audio Free is a WordPress plugin that lets you play your audio files on your site. Create a player in the WordPress Dashboard and add tracks. Supported file types include MP3, OGG, and WAV.

== Description ==
Create an publish players with myriad options including playlist looping, two themes (light and dark), and one player. Player and track information can be updated in the admin page of the WordPress Dashboard. Use the player to showcase music, podcasts, or for any other audio application you can imagine!

== Installation ==
The installation of both the free and pro versions follow the same steps:

1. Download the zip file

2. Go to Plugins → Add New → Upload

3. Click “Install” and then “Activate”

It’s that easy! Now visit the Wafer Audio Free admin page in the WordPress Dashboard to create players.
